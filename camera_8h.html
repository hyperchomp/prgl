Initializes the view (position) and projection matrices for the camera. This sets the camera's position to {0, 0, 0} and sets it as the active camera.

Parameters
cam[in,out]An uninitialized PRGLCamera struct.
fov_degreesThe vertical fov for the camera in degrees. For orthogonal cameras this can be set to zero since it's not needed.
move_speedSpeed value to use when moving the camera.
