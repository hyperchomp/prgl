Runs the core loop of the game.

Handles the core game loop, and calls the given callback functions at the appropriate times. This should be the first function called in your game's main function to ensure prgl is initialized before using it.

prgl_init is called one time before the core game loop starts. All other callbacks are called once per frame in the following order: prgl_update -> prgl_draw_3d -> prgl_draw_2d ->prgl_cleanup`

Parameters
[in]titleTitle of the game, this will be used as the window header
[in]prgl_initInitialize callback, runs one time before the game loop starts.
[in]prgl_updateUpdate callback.
[in]prgl_draw_3dDraws the 3D scene.
[in]prgl_draw_2dDraws a 2D scene on top of the 3D scene.
[in]prgl_cleanupCleanup callback for handling memory at end of frame.
